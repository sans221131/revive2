Build a premium, academic, trust-heavy single-page landing site for Amity University promotion (no course/faculty content). The site must be image-free (no photos, no illustrations). Use only typography, spacing, borders, subtle gradients, and simple SVG line icons.

Visual direction: “Modern Ivy / premium university brochure, but web-clean.” Serious, established, and credible — not flashy. High readability, strong hierarchy, calm luxury.

Color palette:

Primary Navy: #0A2C59

Deep Navy (hero/footer): #041428

Accent Gold (CTAs/highlights): #FACB06

Gold hover: #C8A205

Page background: #FFFFFF

Alt section: #F7F7F7

Border: #E6E8EC

Text: #020203

Muted text: #6C7676

Usage rules:

Navy is the dominant brand color (headers/hero/footers/section titles).

Gold is only for CTA buttons, key highlights, and small accents (never for long paragraphs).

Buttons: Gold background + navy text for primary; navy outline for secondary.

Typography:

Headings: Libre Baskerville (or Merriweather) — weight 700

Body/UI: Inter — weight 400/500

Type feel: authoritative headings + modern readable body.

Layout & spacing:

Max width container: 1200px

Section padding: 72–96px desktop, 48–64px mobile

12-column grid on desktop, stacked on mobile

Cards: white background, 1px border #E6E8EC, subtle shadow, 16–20px radius

UI style:

Components are card-based with clean dividers and consistent spacing.

Use minimal line icons (stroke icons) in navy; gold only as small accent.

Hero uses a subtle deep-navy gradient background with a faint CSS pattern (noise/dots), but keep it classy.

Motion:

Micro-animations only: fade/slide-in on scroll, button hover lift, accordion smooth open/close.

Keep motion subtle and premium (no bouncy stuff).

Tone of copy:

Credible, concise, and confident.

Emphasize trust, recognition, student support, campus life, outcomes, and admissions simplicity.

Avoid hypey marketing slang.
